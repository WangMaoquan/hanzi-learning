# 汉字学习平台 - 设计Moodboard

## 项目概述

- **项目类型**: 儿童教育学习平台
- **目标用户**: 儿童/学生
- **学习内容**: 汉字、古诗、成语、文言文

---

## 风格方向 1: 温馨可爱风格 🎀

### 色彩方案

```
主色:     #FF9E6D (温暖橙色) - 活力、友好
辅色:     #86E3CE (薄荷绿)   - 清新、舒适
强调色:   #FF6B9D (粉红色)   - 活泼、可爱
背景色:   #FFF8F0 (暖白色)   - 柔和、温馨
卡片背景: #FFFFFF           - 干净、清爽
文字主色: #4A4A4A          - 柔和黑色
文字次色: #8A8A8A          - 中性灰
```

### 字体选择

- **标题**: "ZCOOL KuaiLe" (站酷快乐体) - 可爱活泼
- **正文**: "Noto Sans SC" - 清晰易读

### UI 特点

- 圆角: `rounded-2xl` (大圆角)
- 卡片: 柔和阴影 + 圆角边框
- 按钮: 糖果色渐变 + 轻微悬浮效果
- 图标: 圆润填充风格
- 装饰: 云朵、星星、小图案等童趣元素

### 适用场景

适合面向低龄儿童(6-10岁)的启蒙教育产品

## 风格方向 2: 现代简约中国风 🏮

### 色彩方案

```
主色:     #C41E3A (中国红)   - 传统、喜庆
辅色:     #1A1A2E (深靛蓝)  - 沉稳、雅致
强调色:   #D4AF37 (金色)     - 高贵、点缀
背景色:   #FAF8F5 (米白色)  - 宣纸质感
卡片背景: #FFFFFF           - 简洁
文字主色: #2D2D2D          - 清晰黑色
文字次色: #6B6B6B          - 柔和灰
```

### 字体选择

- **标题**: "ZCOOL XiaoWei" (站酷小薇体) - 书法韵味
- **正文**: "Noto Serif SC" - 典雅衬线

### UI 特点

- 圆角: `rounded-lg` (适中圆角)
- 卡片: 细线边框 + 轻微投影
- 按钮: 红色主调 + 金色边框
- 图标: 线性简约风格
- 装饰: 传统纹样、水墨元素

### 适用场景

适合10岁以上青少年及成人，体现文化底蕴

## 风格方向 3: 活泼彩色风格 🌈

### 色彩方案

```
主色:     #6C5CE7 (紫色)     - 创意、想象
辅色:     #00CEC9 (青绿色)   - 科技、活力
强调色:   #FDCB6E (明黄色)   - 欢乐、醒目
背景色:   #F8F9FA (浅灰白)   - 通透
卡片背景: #FFFFFF           - 明亮
文字主色: #2D3436          - 深灰
文字次色: #636E72          - 中灰
渐变:     linear-gradient(135deg, #667eea 0%, #764ba2 100%)
```

### 字体选择

- **标题**: "Poppins" + "Noto Sans SC" - 国际感 + 中文支持
- **正文**: "Noto Sans SC" - 现代无衬线

### UI 特点

- 圆角: `rounded-xl` (较大圆角)
- 卡片: 彩色渐变边框 + 玻璃拟态效果
- 按钮: 渐变背景 + 动效
- 图标: 彩色填充风格
- 装饰: 几何图形、波纹动画

### 适用场景

适合国际化的现代教育产品，突出创新感

## 设计趋势参考 (2025)

从 Awwwards 趋势分析：

1. **大字号标题** - 醒目的 Typography 设计
2. **Bento Grid 布局** - 模块化卡片展示
3. **微妙动画** - 悬浮、过渡效果
4. **玻璃拟态** - 毛玻璃背景效果
5. **渐变应用** - 柔和色彩渐变

---

## 推荐选择

考虑到 **儿童教育平台** 的定位，我推荐：

### 首选: 风格方向 1 (温馨可爱风格)

理由:

- 温暖色调适合低龄学习者
- 大圆角设计更具亲和力
- 活泼的配色能吸引儿童注意力
- 保持专业教育产品形象

### 次选: 风格方向 2 (现代简约中国风)

如果目标用户年龄偏大或需要体现文化底蕴
